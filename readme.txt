This plugin disallow users going to the backend. They can manage theme prifile on the frontend page.

== Description ==

This plugin will disallow users manage them  profile on the admin page. They will have to do that on the frontend page.
Activate the plugin and it creates the profile page with shortcode, and after that users will be redirected to this page if they will want to edit profile. Also you able to use the shortcode on any page, just insert the shortcode - [oi_frontend_profile].

In the next version of the plugin there will be an option page and you will be able to change profile page.

Supported laguages:
<ul>
<li>English</li>
<li>Russian</li>
</ul>

== Installation ==

1. Upload `oi-frontend-profile` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
